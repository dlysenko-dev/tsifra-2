# Phase 1: Quick Landscape Scan — Digital Clone v3

## Макро-обзор (14 мая 2026)

### Key Findings

1. **AI Agent Frameworks взорвали рынок** — AutoGPT (170K stars), CrewAI (28K), Hermes (60K за 2 месяца), LangGraph (24.8K). MCP становится стандартом интеграции инструментов [^8^][^10^][^12^]
2. **Бесплатные решения реально работают** — Ollama + Gemma 4, GLM 5.1 (бесплатный), DeepSeek ($0.27/1K), n8n self-hosted (бесплатно), Dify (open-source) [^13^]
3. **Multi-agent orchestration** — CrewAI (ролевые агенты), LangGraph (stateful workflows), Hermes (persistent memory + self-learning) [^8^][^10^]
4. **Контроль качества** — fact-checking с цитатами (Writesonic), plagiarism checker (Rytr), AI detection tools (ZeroGPT, GPTZero) — но все платные [^9^][^11^]
5. **Telegram боты для бизнеса** — Klipy ($39/мес), n8n (self-hosted бесплатно), Zapier-интеграции. Кастомные боты — open source [^14^][^16^]
6. **Монетизация агентов** — usage-based, subscription tiers, outcome-based pricing, hybrid models. Целевая модель: $29-499/мес [^15^][^20^]
7. **Hermes Agent** — самый амбициозный: persistent multi-layer memory, continuous self-learning, 24/7 cloud-native, enterprise messaging. 60K stars за 2 месяца [^8^]

### Dominant Narratives
- "Free open-source agents match or surpass paid options" [^13^]
